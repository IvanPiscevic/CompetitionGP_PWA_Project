Programiranje Web Aplikacija Projekt - CompetitionGP
Autor: Ivan Piščević 02409471

Projekt je napravljen s php-om i mysql bazom podataka.
U github repozitoriju se nalazi direktorij projekta i baza podataka.

Upute:
1. Direktorij stranice se treba smjestiti unutar htdocs direktorija, unutar xampp direktorija. 
   (npr. C:\xampp\htdocs\CompetitionGP)
2. Baza podataka se importa kroz phpmyadmin sučelje. 
   (Drag and drop sql dokument na export stranici, zatim pritisnite "import" gumb na dnu stranice).
